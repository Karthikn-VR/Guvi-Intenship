# GUVI Project

A minimal black-and-white pixel-styled GUVI landing platform built using HTML, CSS, JavaScript, jQuery, PHP, MySQL. The project includes a responsive landing page, login/register system, and user dashboard. Authentication and profile updates are handled using jQuery AJAX with prepared statements in MySQL.

## Quick Start

- Open `index.html` in your browser to view the homepage
- Use the navbar:
  - Explore → opens the official GUVI site
  - About → opens the About page with project details
  - Login → opens the authentication page

## Pages

- `index.html` — Homepage with hero card and retro feature cards
- `about.html` — Reviewer‑focused documentation in a distinct retro layout
- `auth.html` — Authentication and dashboard UI

## Styling

- `css/retro.css` — Homepage and About page styles (pixel typography, borders, shadows)
- `css/style.css` — Auth page styles

## Backend

- `php/` — Lightweight PHP endpoints for login, registration, profile, and dashboard JSON
- `db_setup.php` — Database initializer (configure securely for production)
